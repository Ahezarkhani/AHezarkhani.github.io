15-112: Fundamentals of Programming

A technical introduction to the fundamentals of programming. We will cover a large subset of the Python programming language. We will also target numerous deployment scenarios, including standalone programs, shell scripts, and web-based applications.

Requirements:
This course assumes no prior programming experience.

Introduction to Front-end Web Development

In this course, you'll learn how to create the same web-pages that inhabit most of the internet. We'll learn HTML, CSS, Bootstrap, and the basics of Javascript and JQuery.

Requirements:
This course assumes no prior programming experience.

Course Page

Introduction to Back-end Web Development

In this course, you'll learn how the internet works and when you're done, you'll be able to contribute to it. You'll learn the Django framework (a popular python framework used by instagram, pinterest and other popular web-sites).

Requirements:
This course assumes knowledge of Python (equal to 15-112)

Course Page

Advanced Front-end Web Development

In this course, we'll cover advanced tools used for front-end web development. We will go deeper into Javascript and JQuery, then jump into React, SASS and Redux.

Requirements:
This course assumes knowledge of front-end web development (equal to Introduction to Front-end Web Development)

Course Page

Advanced Back-end Web Development

This course will build off the Introduction to Back-end Web Development. We will learn how to build robust web applications with Django-Rest, the Django testing framework, deploying to Heroku, AWS, and Google Cloud platform, and other cloud tools.

Requirements:
This course assumes knowledge of back-end web development with Django (equivalent to Introduction to Back-end Web Development)

Course Page

Version Control with Git

In this course, we'll learn how to manage our code base with a popular tool called Git. This will allow us to manage our own code as well as to work in groups. We will also use Github as a GUI for Git.

Requirements:
This course assumes no prior programming experience.
